## Concept

So far in the evolution of AI application development, it's been difficult to build software that dependably and deterministically allows you to ask questions to extract information from your data.

Infactory makes this easy, with tools to deliver **natural language in and structured data out**.

Here's how Infactory technology works.

### Infactory always starts with your data 
Infactory doesn't scrape the web. Dependable and deterministic answers are rooted in known data. Data you have direct access to. Data you're interested in asking questions about. Data you want to use with Infactory technology.

Let's use this CSV file as an example.

### Datalines model your data

**Datalines** are exclusive to Infactory. They place a semantic layer on top of your data, providing consistent naming, data types, structure, and interconnections.

Infactory offers a `generate-dataline` tool to auto-create a Dataline from your data. This always gives you a solid head start, and you can use that Dataline as is. You also can customize the Dataline as you see fit, using the documentated API.

<Accordion title="Example Dataline">
#### Dataline generated from the CSV file

Created from the `nyc_taxi_rides.csv` file using the `generate-dataline` tool.

<CodeGroup>
```bash python
#
# Generated dataline from file: nyc_taxi_rides.csv
#

from infactory import Column, ColumnT, GeneralColumnT, NameT, Schema, SchemaT, NameT, RoleT, Role, TypeT

class NYCTaxiRidesColumnT(ColumnT):
    # Use GeneralColumnT.ID
    PICKUP_DATETIME = Column(
        type_t=TypeT.DATETIME,
        names=(
            (NameT.CANONICAL, "Pickup Datetime"),
            (NameT.DATA_KEY, "pickup_datetime")
        )
    ).with_output_formatter(GeneralColumnT.DATETIME_FORMATTER)

    DROPOFF_DATETIME = Column(
        type_t=TypeT.DATETIME,
        names=(
            (NameT.CANONICAL, "Dropoff Datetime"),
            (NameT.DATA_KEY, "dropoff_datetime")
        )
    ).with_output_formatter(GeneralColumnT.DATETIME_FORMATTER)

    TRIP_DURATION = Column(
        type_t=TypeT.DURATION,
        names=(
            (NameT.CANONICAL, "Trip Duration"),
            (NameT.DATA_KEY, "trip_duration")
        )
    ).with_output_formatter(GeneralColumnT.DURATION_FORMATTER)

    PASSENGER_COUNT = Column(
        type_t=TypeT.ORDINAL,
        names=(
            (NameT.CANONICAL, "Passenger Count"),
            (NameT.DATA_KEY, "passenger_count")
        )
    )

    TRIP_DISTANCE_IN_MILES = Column(
        type_t=TypeT.NUMBER,
        names=(
            (NameT.CANONICAL, "Trip Distance in Miles"),
            (NameT.DATA_KEY, "trip_distance_in_miles")
        )
    )

    PAYMENT_TYPE = Column(
        type_t=TypeT.STRING,
        names=(
            (NameT.CANONICAL, "Payment Type"),
            (NameT.DATA_KEY, "payment_type")
        )
    )

    BASE_FARE = Column(
        type_t=TypeT.NUMBER,
        names=(
            (NameT.CANONICAL, "Base Fare"),
            (NameT.DATA_KEY, "base_fare")
        )
    )

    TAXES = Column(
        type_t=TypeT.NUMBER,
        names=(
            (NameT.CANONICAL, "Taxes"),
```

### Query Programs are the differentiator
The `ask` tool in the previous step builds a **Query Program** based on your question, which then runs and returns a structured data response.

Query Programs are deterministic and repeatable. The structured data returned is always the same as long as the Query Program is the same and the data remains the sameâ€”a crucial characteristic not available in other AI systems and approaches.
<Accordion title="Example Query Program">
The `ask` tool Query Program that answered the question in the previous step:

**Q: What is the average tip for rides with 1, 2, and 3 passengers?**
<CodeGroup>
```bash python
from infactory import *
from taxi.nyc_taxi_rides.nyc_taxi_rides_dataline import *

# Q: What is the average tip for rides with 1, 2, and 3 passengers?
NC = NYCTaxiRidesColumnT
q = (QueryProgram()
    | PushWorkspace('taxi')
    | PushDataFrame(NYCTaxiRidesSchemaT.NYC_TAXI_RIDES)
    | Filter(Expr(NC.PASSENGER_COUNT, ExprT.GTE, 1) & Expr(NC.PASSENGER_COUNT, ExprT.LTE, 3))
    | Agg(AggT.MEAN, NC.TIP) // NC.PASSENGER_COUNT
    | Result(ResultT.MAIN)
)
result = q.run() 
```   
</CodeGroup>
